# Gerenciador de Despesas Pessoais em Python

Um app simples de linha de comando (CLI) pra controlar seus gastos diários. Registre despesas com descrição, categoria, valor e data, liste tudo e veja totais por categoria. Perfeito pra quem quer começar a organizar o dinheiro sem complicação!

Feito como meu primeiro projeto modular em Python, pra praticar OOP, imports entre arquivos, JSON pra salvar dados e tratamento de erros.

## Funcionalidades
- Adicionar despesa (ex: "Dogão no calçadão", categoria "Alimentação", R$ 18.00)
- Listar todas as despesas com total geral
- Ver gastos agrupados por categoria (ex: Alimentação: R$ 150.00)
- Dados salvos em `despesas.json` (não perde nada ao fechar o programa)

## Tecnologias usadas
- Python 3
- Módulos nativos: json, datetime
- Conceitos: Classes (OOP), funções separadas, try/except, list comprehension, imports modulares
